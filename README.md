# #49 Anger Anchor

**Anger Anchor** is a hyper-minimalist 6-second countdown tool designed to intercept emotional outbursts and restore logical thinking using cognitive distraction.

## Core Philosophy
- **1 Tool / 1 Action**: One giant button to drop the "anchor" of reason the moment you feel anger.
- **Cognitive Shift**: Uses simple arithmetic problems to force the brain to switch from the emotional amygdala to the logical prefrontal cortex.
- **6-Second Rule**: Based on neurobiological evidence that the peak of an adrenaline surge lasts approximately 6 seconds.

## Features
- **Instant Activation**: Large, easy-to-tap circular button for high-stress moments.
- **Logic Tasks**: Random math problems generated every second to keep the brain engaged.
- **Visual Feedback**: Graduated background color shifts (Red → Yellow → White) to visually signal the "cooling down" process.
- **Mindful Exit**: Randomly displays one of five grounding affirmations upon completion to ensure a calm transition back to reality.
- **Privacy First**: No login, no tracking, no ads. Operates entirely within your browser.

## Technical Standards
- **iOS Optimized**: Specific CSS to prevent input/text dimming on Safari.
- **High Performance**: Zero dependencies, lightweight vanilla JavaScript.
- **Fridge Combo UI**: Minimalist white background, #333 main elements, and 15px rounded corners for secondary elements.
